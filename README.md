# CODSOFT-Task3
Student Information:
Bandaru Pavani is a 2nd Year B.Tech CSE student and aspiring Software Engineer. She is interested in coding, web development, and learning new technologies.

Project Overview:
The Modern Calculator is a responsive web application created using HTML, CSS, and JavaScript. It performs basic arithmetic operations like addition, subtraction, multiplication, division, percentage calculations, and provides an interactive user-friendly interface.

Objective:
To create a functional calculator using front-end technologies.
To improve JavaScript programming skills.
To understand DOM manipulation and event handling.
To design a responsive and attractive user interface.

Features:
 Responsive calculator design
 Interactive buttons
 Addition, subtraction, multiplication, and division operations
 Percentage calculation
 Clear screen option
 Delete last character option
 Hover effects and animations
 Error handling for invalid calculations

Technologies Used:
 HTML5
 CSS3
 JavaScript
 CSS Grid
 Flexbox

Functions/Concepts Used:
 DOM Manipulation
 JavaScript Functions
 Event Handling
 CSS Grid Layout
 Hover Effects
 Transitions
 Responsive Design
 Arithmetic Operations using eval()

Advantages:
 Easy to use interface
 Fast calculations
 Mobile-friendly design
 Improves JavaScript practice
 Attractive modern UI

Applications:
Basic mathematical calculations
Student mini project
Internship task submission
JavaScript practice project
Beginner web development project

Conclusion:
This project helps in understanding the fundamentals of front-end web development using HTML, CSS, and JavaScript. It improves problem-solving skills and provides practical experience in building interactive web applications.
